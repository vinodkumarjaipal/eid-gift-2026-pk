# 🌙 Eid Gift 2026 - Digital Eidi Rewards Platform 💸

## 🌟 Overview
**Eid Gift 2026** is a high-end web application designed to bring joy this Eid. Users can earn digital rewards by watching ads, inviting friends, and completing simple tasks. Built with a focus on security and a premium **Glassmorphism UI**.

## ✨ Features
* **💎 Glassmorphism UI:** Modern, sleek, and dark-themed interface.
* **🔒 Anti-Cheat System:** Advanced IP-locking and Device Fingerprinting to prevent multiple accounts.
* **📱 Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.
* **💰 Earning Modules:** * **Watch & Earn:** Get Rs. 15 for every 30-second ad.
    * **Referral System:** Earn Rs. 50 for every successful invite.
* **🏧 Withdrawal System:** Integrated request forms for JazzCash and EasyPaisa.
* **⚡ Real-time Sync:** Powered by Firebase Firestore for instant balance updates.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Tailwind-ready animations), JavaScript (ES6+).
* **Backend:** Firebase (Firestore Database).
* **Ads Integration:** Monetag SmartLink & Vignette Ads.
* **Security:** Device Fingerprinting (Base64 Encoding).
